# PeladaPay API

API .NET 9 com Clean Architecture, CQRS (MediatR), EF Core/PostgreSQL, Identity + JWT, FluentValidation e Swagger.

## Subir com Docker

```bash
docker compose up --build
```

A API sobe em `http://localhost:8080` e o Swagger em `/swagger`.
